### Assertions

```js
const assert = require('assert')
```

```js
assert(val)
assert.equal(actual, expected)
assert.notEqual(a, e)
```

```js
assert.deepEqual(a, e)
assert.notDeepEqual(a, e)
```

```js
assert.throws(fn)
```
