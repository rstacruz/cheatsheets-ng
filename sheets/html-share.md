## Share links

<!-- {.-one-column} -->

#### Facebook

```html
<a href='https://www.facebook.com/sharer/sharer.php?u=URL' target='share'>
```

#### Twitter

```html
<a href='https://twitter.com/intent/tweet?text=DESCRIPTION+URL' target='share'>
```

#### Google Plus

```html
<a href='https://plus.google.com/share?url=URL' target='share'>
```
